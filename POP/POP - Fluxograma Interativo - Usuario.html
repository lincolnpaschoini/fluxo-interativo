1. Objetivo

Padronizar a utilização do Sistema de Fluxograma Interativo pelos colaboradores da Paschoini Advogados, estabelecendo procedimentos claros para acesso, navegação, edição de conteúdo, solicitação de permissões e salvamento das informações registradas nos processos operacionais da empresa.

2. Aplicabilidade

Aplica-se a todos os colaboradores com perfil de Usuário Comum que utilizam o Sistema de Fluxograma Interativo para consulta e atualização dos processos operacionais. Usuários com perfil de Administrador possuem procedimento específico complementar.

3. Áreas Envolvidas

  • a) Usuário Operacional: navegar, consultar e editar os sub-fluxos autorizados pelo Administrador, mantendo as informações sempre atualizadas.
  • b) Gestor da Área: orientar o uso correto da ferramenta, validar o conteúdo registrado e solicitar acesso às caixas necessárias.
  • c) Administrador do Sistema: cadastrar usuários, conceder permissões de edição e garantir a estabilidade do sistema.

4. Referências

  • Procedimentos internos de operação e segurança da informação da Paschoini Advogados.
  • Documentação técnica do Sistema de Fluxograma Interativo.
  • Políticas de uso de sistemas corporativos da empresa.

5. Premissas

  • O usuário deve possuir conta Microsoft corporativa (@paschoini.adv.br) ativa para autenticação no sistema.
  • O acesso para edição de cada caixa do fluxo é restrito e deve ser solicitado e aprovado pelo Administrador do Sistema.
  • Todas as alterações realizadas são salvas automaticamente e ficam registradas no histórico de auditoria.
  • As imagens inseridas são armazenadas de forma permanente no servidor — não são perdidas em atualizações do sistema.
  • O sistema pode ser acessado por computador ou dispositivo móvel por qualquer navegador moderno (Chrome, Edge, Firefox).
  • Em caso de dúvidas ou problemas técnicos, acionar o Administrador do Sistema.

6. Procedimento

6.1. Acessar o Sistema (Login)

  1. 6.1.1.Abrir o navegador (Chrome, Edge ou Firefox) e acessar o endereço do sistema: [INSERIR URL DO SISTEMA].
  2. 6.1.2.Na tela de login, clicar no botão "Entrar com Microsoft".
  3. 6.1.3.Inserir o e-mail corporativo (@paschoini.adv.br) e a senha da conta Microsoft quando solicitado.
  4. 6.1.4.Após a autenticação, o fluxograma principal será exibido automaticamente.
  5. 6.1.5.Caso o acesso seja negado ou ocorra erro, acionar o Administrador do Sistema.

6.2. Navegar no Fluxograma

  1. 6.2.1.Ao acessar o sistema, o fluxograma principal é exibido com todas as caixas de processo.
  2. 6.2.2.Para mover o fluxo: clicar e arrastar com o botão esquerdo do mouse em área vazia.
  3. 6.2.3.Para aplicar zoom: utilizar o scroll do mouse (rolar para frente = aproximar; para trás = afastar).
  4. 6.2.4.Em dispositivos móveis: deslizar com um dedo para mover o fluxo; usar dois dedos (pinça) para zoom.
  5. 6.2.5.Clicar sobre qualquer caixa do fluxo para abrir o sub-fluxo correspondente.
📷 Fluxograma Principal Clique aqui para selecionar, depois Ctrl+V para colar o print

6.3. Solicitar Acesso a uma Caixa

Atenção: Apenas caixas com permissão liberada pelo Administrador permitem edição. Ao abrir uma caixa sem permissão, o sistema exibirá uma mensagem de restrição.
  1. 6.3.1.Clicar sobre a caixa do fluxo que deseja editar para abrir o sub-fluxo.
  2. 6.3.2.Caso não possua permissão, o sistema exibirá a mensagem com o botão "Solicitar Acesso".
  3. 6.3.3.Clicar no botão "Solicitar Acesso" e aguardar a confirmação de envio.
  4. 6.3.4.Aguardar o Administrador aprovar — uma notificação será exibida automaticamente na tela.
  5. 6.3.5.Após aprovação, recarregar a página (F5) para ativar o acesso de edição.

6.4. Abrir e Visualizar um Sub-fluxo

  1. 6.4.1.Clicar sobre a caixa desejada no fluxograma principal.
  2. 6.4.2.O sub-fluxo será aberto exibindo as etapas do processo em sequência horizontal.
  3. 6.4.3.Cada etapa exibe: título, descrição, responsável, duração e eventuais imagens e links.
  4. 6.4.4.Para fechar, clicar em "← Fechar" ou pressionar Esc.
📷 Sub-fluxo Aberto com Etapas Clique aqui para selecionar, depois Ctrl+V para colar o print

6.5. Editar Etapas do Sub-fluxo

Dica: As alterações são sincronizadas em tempo real para todos os usuários conectados.
  1. 6.5.1.Com o sub-fluxo aberto, localizar a etapa que deseja editar.
  2. 6.5.2.Clicar diretamente sobre o título da etapa para editá-lo.
  3. 6.5.3.Clicar na área de descrição para adicionar ou alterar o conteúdo (aceita negrito, itálico, listas).
  4. 6.5.4.Preencher o campo "Responsável" com o nome do colaborador responsável pela etapa.
  5. 6.5.5.Preencher o campo "Duração" com o tempo estimado (ex.: "2 dias", "1 hora").
  6. 6.5.6.Para adicionar nova etapa, clicar no botão "+ Etapa" ao final da lista.
  7. 6.5.7.Para remover uma etapa, clicar no ícone de exclusão (lixeira). Atenção: ação irreversível.
📷 Etapa em Modo de Edição Clique aqui para selecionar, depois Ctrl+V para colar o print

6.6. Adicionar Imagem a uma Etapa

  1. 6.6.1.Com o sub-fluxo aberto, localizar a etapa desejada.
  2. 6.6.2.Clicar no ícone de imagem / câmera dentro da etapa.
  3. 6.6.3.Selecionar o arquivo de imagem no computador. Formatos aceitos: JPG, PNG, GIF, WEBP.
  4. 6.6.4.Aguardar o carregamento. A imagem será exibida em miniatura dentro da etapa.
  5. 6.6.5.Clicar sobre a miniatura para ampliar a imagem em tela cheia.
  6. 6.6.6.Para remover, clicar no ícone de exclusão sobre a miniatura da imagem.
  7. 6.6.7.É possível adicionar múltiplas imagens em uma mesma etapa.
📷 Botão de Adição de Imagem Clique aqui para selecionar, depois Ctrl+V para colar o print
📷 Etapa com Imagem Inserida Clique aqui para selecionar, depois Ctrl+V para colar o print

6.7. Adicionar Link a uma Etapa

  1. 6.7.1.Com o sub-fluxo aberto, localizar a etapa desejada.
  2. 6.7.2.Clicar no botão "Adicionar Link" dentro da etapa (ícone de corrente).
  3. 6.7.3.No campo "Nome", digitar um texto descritivo (ex.: "Proposta Comercial", "Planilha de Controle").
  4. 6.7.4.No campo "URL", colar o endereço completo do link (ex.: https://docs.google.com/...).
  5. 6.7.5.O link será salvo e exibido na etapa, clicável por todos os usuários.
  6. 6.7.6.Para remover, clicar no ícone de exclusão ao lado do link cadastrado.

6.8. Adicionar Terceiro Nível (Sub-etapas)

Atenção: O terceiro nível permite detalhar uma etapa em passos menores. Use quando a etapa envolver múltiplas ações distintas.
  1. 6.8.1.Com o sub-fluxo aberto, localizar a etapa que deseja detalhar.
  2. 6.8.2.Clicar no botão "+ Sub-etapas" ou no ícone de terceiro nível dentro da etapa.
  3. 6.8.3.Um novo painel será aberto exibindo as sub-etapas daquela etapa.
  4. 6.8.4.Para adicionar nova sub-etapa, clicar em "+ Sub-etapa".
  5. 6.8.5.Preencher o título e a descrição de cada sub-etapa, e o responsável e duração se necessário.
  6. 6.8.6.Para voltar ao sub-fluxo principal, clicar em "← Voltar".
  7. 6.8.7.A etapa que possui sub-etapas será marcada visualmente com um indicador especial.

6.9. Salvar Alterações

Dica: Alterações nos sub-fluxos são sincronizadas automaticamente. O botão "Salvar" garante a persistência do estado geral do fluxo.
  1. 6.9.1.As alterações nos sub-fluxos são sincronizadas automaticamente ao serem realizadas.
  2. 6.9.2.Ao concluir as edições, clicar no botão "Salvar" na barra superior do sistema.
  3. 6.9.3.Aguardar a confirmação visual: o botão exibirá o status "Salvo ✓".
  4. 6.9.4.Em caso de falha, verificar a conexão com a internet e tentar novamente.
  5. 6.9.5.Todas as alterações ficam registradas no histórico de auditoria e podem ser consultadas pelo Administrador.
